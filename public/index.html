旅游日记平台 API

这是旅游日记平台的后端 API 服务，提供用户系统和游记管理功能。

主要功能

  • 用户注册和登录
  • 游记发布、查看和管理
  • 内容审核系统

API 文档

可用的 API 端点：

GET /api/diaries

获取所有已批准的游记列表（分页）

GET /api/diaries/:id

获取特定游记的详细信息

POST /api/users/register

用户注册

POST /api/users/login

用户登录

更多详细的 API 文档正在整理中...

技术栈

  • Node.js + Express
  • MongoDB + Mongoose
  • JWT 认证
